Application

This is presentation pitch for the final project of Developing Data Products.

Developing Data Products Project

The project is a proof of concept and pretend to be a simple Shiny Application to explore the Stackloss dataset, also there is an implmentation of a linear model and a plot that display the variables and the relationship among them.

Dataset

The dataset used is stackloss (Brownlee’s Stack Loss Plant Data), which is the operational data of a plant for the oxidation of ammonia to nitric acid.

stackloss is a data frame with 21 observations on 4 variables.

  • [,1] Air Flow Flow of cooling air
  • [,2] Water Temp Cooling Water Inlet Temperature
  • [,3] Acid Conc. Concentration of acid [per 1000, minus 500]
  • [,4] stack.loss Stack loss

The app offers to the user the ability to plot the variables and apply a color coding based on a indepent variable.

There’s a feature to subset data (toogle points) and a summary of the linear model.

Application plot

By default, the application will show the variables stack.loss and Air.Flow.

  ggplot(dataset) + aes(x = Air.Flow, y = stack.loss, color = Water.Temp) + geom_point() +
    xlab("Flow of cooling air") + ylab("Stack loss") +
    geom_smooth(method = lm, fullrange = TRUE, color = "black")

Linear Model

In the summary panel, will be appear the linear model for the predictor ~ outcome selected.

  # Calculate linear model
  summary(lm(data = dataset, stack.loss ~ Air.Flow))
